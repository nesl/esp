iTunes.app terminology

/Applications/iTunes.app

Standard Suite

Common terms for most applications

Commands

  • reference.print_(...) -- Print the specified object(s)
    • [print_dialog=Boolean] -- Should the application show the print dialog
    • [with_properties=k.print_settings] -- the print settings
    • [kind=k.track_listing | k.album_listing | k.cd_insert] -- the kind of printout desired
    • [theme=Unicode] -- name of theme to use for formatting the printout
    • Result: None
  • reference.close() -- Close an object
    • Result: None
  • reference.count(...) -- Return the number of elements of a particular class within an object
    • each=Type -- the class of the elements to be counted. Keyword 'each' is optional in AppleScript
    • Result: Integer -- the number of elements
  • reference.delete() -- Delete an element from an object
    • Result: None
  • reference.duplicate(...) -- Duplicate one or more object(s)
    • [to=InsertionRef] -- the new location for the object(s)
    • Result: Reference -- to the duplicated object(s)
  • reference.exists() -- Verify if an object exists
    • Result: Boolean -- true if it exists, false if not
  • application.make(...) -- Make a new element
    • new=Type -- the class of the new element. Keyword 'new' is optional in AppleScript
    • [at=InsertionRef] -- the location at which to insert the element
    • [with_properties=Record] -- the initial values for the properties of the element
    • Result: Reference -- to the new object(s)
  • application.move(...) -- Move playlist(s) to a new location
    • k.playlist -- the playlist(s) to move
    • to=InsertionRef -- the new location for the playlist(s)
    • Result: None
  • reference.open() -- open the specified object(s)
    • Result: None
  • application.run() -- run iTunes
    • Result: None
  • application.quit() -- quit iTunes
    • Result: None
  • reference.set(...)
    • to=Anything
    • Result: None

Classes

  • print_settings
    • Properties:
      • copies (r/o) Integer -- the number of copies of a document to be printed
      • collating (r/o) Boolean -- Should printed copies be collated?
      • starting_page (r/o) Integer -- the first page of the document to be printed
      • ending_page (r/o) Integer -- the last page of the document to be printed
      • pages_across (r/o) Integer -- number of logical pages laid across a physical page
      • pages_down (r/o) Integer -- number of logical pages laid out down a physical page
      • error_handling (r/o) k.standard | k.detailed -- how errors are handled
      • requested_print_time (r/o) DateTime -- the time at which the desktop printer should print the document
      • printer_features (r/o) List -- printer specific options
      • fax_number (r/o) String -- for fax number
      • target_printer (r/o) String -- for target printer

iTunes Suite

The event suite specific to iTunes

Commands

  • application.add(...) -- add one or more files to a playlist
    • list of Alias -- the file(s) to add
    • [to=InsertionRef] -- the location of the added file(s)
    • Result: k.track -- reference to added track(s)
  • application.back_track() -- reposition to beginning of current track or go to previous track if already at start of current track
    • Result: None
  • reference.convert() -- convert one or more files or tracks
    • Result: k.track -- reference to converted track(s)
  • application.fast_forward() -- skip forward in a playing track
    • Result: None
  • application.next_track() -- advance to the next track in the current playlist
    • Result: None
  • application.pause() -- pause playback
    • Result: None
  • reference.play(...) -- play the current track or the specified track or file.
    • [once=Boolean] -- If true, play this track once and then stop.
    • Result: None
  • application.playpause() -- toggle the playing/paused state of the current track
    • Result: None
  • application.previous_track() -- return to the previous track in the current playlist
    • Result: None
  • application.refresh(...) -- update file track information from the current information in the track’s file
    • k.file_track -- the file track to update
    • Result: None
  • application.resume() -- disable fast forward/rewind and resume playback, if playing.
    • Result: None
  • application.rewind() -- skip backwards in a playing track
    • Result: None
  • application.search(...) -- search a playlist for tracks matching the search string. Identical to entering search text in the Search field in iTunes.
    • k.playlist -- the playlist to search
    • for_=Unicode -- the search text
    • [only=k.albums | k.all | k.artists | k.composers | k.displayed | k.songs] -- area to search (default is all)
    • Result: k.track -- reference to found track(s)
  • application.stop() -- stop playback
    • Result: None
  • application.update(...) -- update the specified iPod
    • [k.source] -- the iPod to update
    • Result: None
  • application.eject(...) -- eject the specified iPod
    • [k.source] -- the iPod to eject
    • Result: None
  • application.subscribe(...) -- subscribe to a podcast feed
    • String -- the URL of the feed to subscribe to
    • Result: None
  • application.updateAllPodcasts() -- update all subscribed podcast feeds
    • Result: None
  • application.updatePodcast() -- update podcast feed
    • Result: None
  • application.download(...) -- download podcast episode
    • k.URL_track
    • Result: None

Classes

  • application -- The application program
    • Properties:
      • current_encoder k.encoder -- the currently selected encoder (MP3, AIFF, WAV, etc.)
      • current_EQ_preset k.EQ_preset -- the currently selected equalizer preset
      • current_playlist (r/o) k.playlist -- the playlist containing the currently targeted track
      • current_stream_title (r/o) Unicode -- the name of the current song in the playing stream (provided by streaming server)
      • current_stream_URL (r/o) Unicode -- the URL of the playing stream or streaming web site (provided by streaming server)
      • current_track (r/o) k.track -- the current targeted track
      • current_visual k.visual -- the currently selected visual plug-in
      • EQ_enabled Boolean -- is the equalizer enabled?
      • fixed_indexing Boolean -- true if all AppleScript track indices should be independent of the play order of the owning playlist.
      • frontmost Boolean -- is iTunes the frontmost application?
      • full_screen Boolean -- are visuals displayed using the entire screen?
      • name (r/o) Unicode -- the name of the application
      • mute Boolean -- has the sound output been muted?
      • player_position Integer -- the player’s position within the currently playing track in seconds.
      • player_state (r/o) k.stopped | k.playing | k.paused | k.fast_forwarding | k.rewinding -- is iTunes stopped, paused, or playing?
      • selection (r/o) Reference -- the selection visible to the user
      • sound_volume Integer -- the sound output volume (0 = minimum, 100 = maximum)
      • version (r/o) Version -- the version of iTunes
      • visuals_enabled Boolean -- are visuals currently being displayed?
      • visual_size k.small | k.medium | k.large -- the size of the displayed visual
    • Elements:
      • browser_windows -- index | name | id
      • encoders -- index | name | id
      • EQ_presets -- index | name | id
      • EQ_windows -- index | name | id
      • playlist_windows -- index | name | id
      • sources -- index | name | id
      • visuals -- index | name | id
      • windows -- index | name | id
  • artwork -- a piece of art within a track
    • Parent:
      • item
    • Properties:
      • format (r/o) Type -- the data format for this piece of artwork
      • data Pict -- data for this artwork, in the form of a picture
      • kind Integer -- kind or purpose of this piece of artwork
  • audio_CD_playlist -- a playlist representing an audio CD
    • Parent:
      • playlist
    • Properties:
      • artist Unicode -- the artist of the CD
      • compilation Boolean -- is this CD a compilation album?
      • composer Unicode -- the composer of the CD
      • disc_count Integer -- the total number of discs in this CD’s album
      • disc_number Integer -- the index of this CD disc in the source album
      • genre Unicode -- the genre of the CD
      • year Integer -- the year the album was recorded/released
    • Elements:
      • audio_CD_tracks -- index | name | id
  • audio_CD_track -- a track on an audio CD
    • Parent:
      • track
    • Properties:
      • location (r/o) Alias -- the location of the file represented by this track
  • browser_window -- the main iTunes window
    • Parent:
      • window
    • Properties:
      • minimized Boolean -- is the small player visible?
      • selection (r/o) Reference -- the selected songs
      • view k.playlist -- the playlist currently displayed in the window
  • device_playlist -- a playlist representing the contents of a portable device
    • Parent:
      • playlist
    • Elements:
      • device_tracks -- index | name | id
  • device_track -- a track residing on a portable music player
    • Parent:
      • track
  • encoder -- converts a track to a specific file format
    • Parent:
      • item
    • Properties:
      • format (r/o) Unicode -- the data format created by the encoder
  • EQ_preset -- equalizer preset configuration
    • Parent:
      • item
    • Properties:
      • band_1 Float -- the equalizer 32 Hz band level (-12.0 dB to +12.0 dB)
      • band_2 Float -- the equalizer 64 Hz band level (-12.0 dB to +12.0 dB)
      • band_3 Float -- the equalizer 125 Hz band level (-12.0 dB to +12.0 dB)
      • band_4 Float -- the equalizer 250 Hz band level (-12.0 dB to +12.0 dB)
      • band_5 Float -- the equalizer 500 Hz band level (-12.0 dB to +12.0 dB)
      • band_6 Float -- the equalizer 1 kHz band level (-12.0 dB to +12.0 dB)
      • band_7 Float -- the equalizer 2 kHz band level (-12.0 dB to +12.0 dB)
      • band_8 Float -- the equalizer 4 kHz band level (-12.0 dB to +12.0 dB)
      • band_9 Float -- the equalizer 8 kHz band level (-12.0 dB to +12.0 dB)
      • band_10 Float -- the equalizer 16 kHz band level (-12.0 dB to +12.0 dB)
      • modifiable (r/o) Boolean -- can this preset be modified?
      • preamp Float -- the equalizer preamp level (-12.0 dB to +12.0 dB)
      • update_tracks Boolean -- should tracks which refer to this preset be updated when the preset is renamed or deleted?
  • EQ_window -- the iTunes equalizer window
    • Parent:
      • window
    • Properties:
      • minimized Boolean -- is the small EQ window visible?
  • file_track -- a track representing an audio file (MP3, AIFF, etc.)
    • Parent:
      • track
    • Properties:
      • location (r/o) Alias -- the location of the file represented by this track
  • folder_playlist -- a folder that contains other playlists
    • Parent:
      • user_playlist
  • item -- an item
    • Children:
      • window
      • track
      • playlist
      • visual
      • encoder
      • EQ_preset
      • artwork
      • source
    • Properties:
      • container (r/o) Reference -- the container of the item
      • name Unicode -- the name of the item
      • persistent_ID (r/o) Long -- the id of the item. This id does not change over time.
  • library_playlist -- the master music library playlist
    • Parent:
      • playlist
    • Elements:
      • file_tracks -- index | name | id
      • URL_tracks -- index | name | id
      • shared_tracks --
  • playlist -- a list of songs/streams
    • Parent:
      • item
    • Children:
      • device_playlist
      • library_playlist
      • user_playlist
      • audio_CD_playlist
      • radio_tuner_playlist
    • Properties:
      • duration (r/o) Integer -- the total length of all songs (in seconds)
      • index (r/o) Integer -- the index of the playlist in internal application order
      • name Unicode -- the name of the playlist
      • parent (r/o) k.playlist -- folder which contains this playlist (if any)
      • shuffle Boolean -- play the songs in this playlist in random order?
      • size (r/o) Long -- the total size of all songs (in bytes)
      • song_repeat k.off | k.one | k.all -- playback repeat mode
      • special_kind (r/o) k.none | k.folder | k.Party_Shuffle | k.Podcasts | k.Purchased_Music | k.Videos -- special playlist kind
      • time (r/o) Unicode -- the length of all songs in MM:SS format
      • visible (r/o) Boolean -- is this playlist visible in the Source list?
    • Elements:
      • tracks -- index | name | id
  • playlist_window -- a sub-window showing a single playlist
    • Parent:
      • window
    • Properties:
      • selection (r/o) Reference -- the selected songs
      • view (r/o) k.playlist -- the playlist displayed in the window
  • radio_tuner_playlist -- the radio tuner playlist
    • Parent:
      • playlist
    • Elements:
      • URL_tracks -- index | name | id
  • shared_track -- a track residing in a shared library
    • Parent:
      • track
  • source -- a music source (music library, CD, device, etc.)
    • Parent:
      • item
    • Properties:
      • capacity (r/o) Long -- the total size of the source if it has a fixed size
      • free_space (r/o) Long -- the free space on the source if it has a fixed size
      • kind (r/o) k.library | k.iPod | k.audio_CD | k.MP3_CD | k.device | k.radio_tuner | k.shared_library | k.unknown
    • Elements:
      • audio_CD_playlists -- index | name | id
      • device_playlists -- index | name | id
      • library_playlists -- index | name | id
      • playlists -- index | name | id
      • radio_tuner_playlists -- index | name | id
      • user_playlists -- index | name | id
  • track -- playable audio source
    • Parent:
      • item
    • Children:
      • shared_track
      • device_track
      • file_track
      • URL_track
      • audio_CD_track
    • Properties:
      • album Unicode -- the album containing the track
      • artist Unicode -- the artist/source of the track
      • bit_rate (r/o) Integer -- the bit rate of the track (in kbps)
      • bookmark Integer -- the bookmark time of the track in seconds
      • bookmarkable Boolean -- is the playback position for this track remembered?
      • bpm Integer -- the tempo of this track in beats per minute
      • category Unicode -- the category of the track
      • comment Unicode -- freeform notes about the track
      • compilation Boolean -- is this track from a compilation album?
      • composer Unicode -- the composer of the track
      • database_ID (r/o) Integer -- the common, unique ID for this track. If two tracks in different playlists have the same database ID, they are sharing the same data.
      • date_added (r/o) DateTime -- the date the track was added to the playlist
      • description Unicode -- the description of the track
      • disc_count Integer -- the total number of discs in the source album
      • disc_number Integer -- the index of the disc containing this track on the source album
      • duration (r/o) Integer -- the length of the track in seconds
      • enabled Boolean -- is this track checked for playback?
      • EQ Unicode -- the name of the EQ preset of the track
      • finish Integer -- the stop time of the track in seconds
      • genre Unicode -- the music/audio genre (category) of the track
      • grouping Unicode -- the grouping (piece) of the track. Generally used to denote movements within a classical work.
      • kind (r/o) Unicode -- a text description of the track
      • long_description Unicode
      • lyrics Unicode -- the lyrics of the track
      • modification_date (r/o) DateTime -- the modification date of the content of this track
      • played_count Integer -- number of times this track has been played
      • played_date DateTime -- the date and time this track was last played
      • podcast (r/o) Boolean -- is this track a podcast episode?
      • rating Integer -- the user rating of this track (0 to 100)
      • sample_rate (r/o) Integer -- the sample rate of the track (in Hz)
      • shufflable Boolean -- is this track included when shuffling?
      • size (r/o) Integer -- the size of the track (in bytes)
      • start_ Integer -- the start time of the track in seconds
      • time (r/o) Unicode -- the length of the track in MM:SS format
      • track_count Integer -- the total number of tracks on the source album
      • track_number Integer -- the index of the track on the source album
      • volume_adjustment Integer -- relative volume adjustment of the track (-100% to 100%)
      • year Integer -- the year the track was recorded/released
    • Elements:
      • artworks -- index
  • URL_track -- a track representing a network stream
    • Parent:
      • track
    • Properties:
      • address Unicode -- the URL for this track
  • user_playlist -- custom playlists created by the user
    • Parent:
      • playlist
    • Children:
      • folder_playlist
    • Properties:
      • shared Boolean -- is this playlist shared?
      • smart (r/o) Boolean -- is this a Smart Playlist?
    • Elements:
      • file_tracks -- index | name | id
      • URL_tracks -- index | name | id
      • shared_tracks --
  • visual -- a visual plug-in
    • Parent:
      • item
  • window -- any window
    • Parent:
      • item
    • Children:
      • playlist_window
      • browser_window
      • EQ_window
    • Properties:
      • bounds QDRectangle -- the boundary rectangle for the window
      • closeable (r/o) Boolean -- does the window have a close box?
      • collapseable (r/o) Boolean -- does the window have a collapse (windowshade) box?
      • collapsed Boolean -- is the window collapsed?
      • position QDPoint -- the upper left position of the window
      • resizable (r/o) Boolean -- is the window resizable?
      • visible Boolean -- is the window visible?
      • zoomable (r/o) Boolean -- is the window zoomable?
      • zoomed Boolean -- is the window zoomed?
